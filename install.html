Install from Eclipse Marketplace

Extras for Eclipse is available in the Eclipse Marketplace. The easiest way to install it is to drag the icon on the start page into your Eclipse.

You can also open the marketplace dialog in Eclipse (Help > Eclipse Marketplace…) and search for "extras".

Update

The software is released continuously, and the sources listed here always contain the latest stable version. See the News page for a summary of the most recent changes.

If installed from the marketplace or software repository, Eclipse finds updates for you: Help > Check for Updates

Uninstall

The feature can be uninstalled as follows: Help > About Eclipse > Installation Details Select Extras for Eclipse and press Uninstall

Software Repository

If you prefer, you can also install Extras for Eclipse directly from this update site:

https://rherrmann.github.io/eclipse-extras/repository/

In the Eclipse main menu, select Help > Install New Software…, then enter the URL above and select Extras for the Eclipse IDE. Expand the item to select only certain features for installation.

A zipped version of the repository can also be downloaded for offline use.

Requirements

A JRE 8 or later and Eclipse Neon (4.6) or later are required to run this software.

Important: If you don't run Eclipse with Java 1.8, the update process may appear to succeed, but Extras for Eclipse will simply not show in the target installation.
